1) Download the zip file and unzip it
2) Run the command "npm install" to install all the angular dependencies
3) Run the command "ng serve --open" to open the page in default browser.
4) Click on "Click Here to upload csv file" and select  the csv file which is to be displayed in the table.
5) Can filter using the "Minimal IssueCount" text box to filter the records depending on minimal issue count
6) For many records, scroll bar is created with fixed table headers for easy visualization.
